### 1. 后端API扩展

- ✅ `GET /api/config` - 获取爬虫配置

### 2. Vue 3 前端项目

#### 项目结构
```
frontend/
├── src/
│   ├── views/
│   │   ├── HomeView.vue        # 首页（系统介绍+状态统计）
│   │   ├── CrawlerView.vue     # 爬虫管理页
│   │   └── QuestionsView.vue   # 面试题生成页
│   ├── services/
│   │   ├── api.js              # Axios实例配置
│   │   └── index.js            # API接口封装
│   ├── router/
│   │   └── index.js            # 路由配置
│   ├── App.vue                 # 主应用组件
│   └── main.js                 # 入口文件
├── index.html
├── vite.config.js              # Vite配置（含代理）
├── package.json
└── README.md
```

#### 技术栈
- Vue 3 (Composition API)
- Vite 5
- Ant Design Vue 4
- Vue Router 4
- Pinia（状态管理）
- Axios（HTTP客户端）
- Day.js（时间处理）

## 🚀 使用方法

### 开发环境
```bash
# 1. 启动后端
python -m app.main

# 2. 启动前端（新终端）
cd frontend
npm install
npm run dev

# 3. 访问 http://localhost:3000
```

### 生产环境
```bash
# 构建前端
cd frontend
npm run build

# 部署 dist 目录到静态服务器
# 配置反向代理转发API请求
```
